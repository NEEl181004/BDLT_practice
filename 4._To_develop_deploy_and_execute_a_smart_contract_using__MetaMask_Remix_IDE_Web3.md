# <a name="_s5helkbwxjgh"></a>**🔹 PART 1: INSTALL & SETUP METAMASK**
## <a name="_25wyv2xtu48q"></a>**Step 1: Install MetaMask**
- Go to:[ ](https://metamask.io/)[https://metamask.io](https://metamask.io/)
- Install Chrome extension
- Add extension to browser
-----
## <a name="_k1vmdeuvejdo"></a>**Step 2: Create Wallet**
- Open MetaMask
- Click **Create Wallet**
- Set password
- Save **12-word recovery phrase (VERY IMPORTANT)**
-----
## <a name="_mjq6lztj229t"></a>**Step 3: Select Sepolia Network**
- Open MetaMask
- Click network dropdown
- Select **Sepolia Test Network**

If not visible:

- Settings → Advanced → Enable test networks
-----
##
## <a name="_vfdnp2og2uep"></a><a name="_rdyuvdugh4z7"></a>**Step 4: Get Test ETH**
- Copy wallet address
- Go to Sepolia Faucet
- Paste address → Request ETH
- Wait for balance
-----

-----

# <a name="_t060rntwm6b3"></a>**🔹 PART 4: DEPLOY CONTRACT**
## <a name="_dk6zkl687alj"></a>**Step 9: Open Deploy Tab**
- Click **Deploy & Run Transactions**
-----
## <a name="_ibqn0cwis87k"></a>**Step 10: Select Environment**
Choose:

Injected Provider - MetaMask

👉 MetaMask popup opens

-----
## <a name="_24ww1knzq6kx"></a>**Step 11: Connect Wallet**
- Click **Next**
- Click **Connect**
- Ensure network = **Sepolia**
-----
## <a name="_i54fvp33fl0e"></a>**Step 12: Deploy Contract**
- Click **Deploy**
- Confirm transaction in MetaMask

✔ Contract deployed successfully

-----
## <a name="_cszy9x3n5rtr"></a>**Step 13: Locate Contract**
Scroll down to:

👉 **Deployed Contracts**

You will see:

SimpleStorage at 0xABC123....

Click arrow (▶) to expand

100
