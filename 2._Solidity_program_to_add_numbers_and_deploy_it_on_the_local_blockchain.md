**Solidity program to add numbers and deploy it on the local
blockchain.**

## **🔴 Step 1: Start Ganache**

-   Open Ganache

-   Click **Quickstart**

-   Keep it running ✅

## **🔴 Step 2: Open your project folder**

cd C:\\Users\\Purva\\MyTruffleProject

## **🔴 Step 3: Create new contract**

cd contracts

notepad Calculator.sol

## **🔴 Step 4: Paste Solidity code**

// SPDX-License-Identifier: UNLICENSED

pragma solidity 0.8.0;

contract Calculator {

uint public result;

function add(uint a, uint b) public {

result = a + b;

}

function getResult() public view returns (uint) {

return result;

}

}

## **🔴 Step 5: Update migration file**

cd ..

cd migrations

notepad 2_deploy_contracts.js

👉 Replace with:

const Calculator = artifacts.require(\"Calculator\");

module.exports = function (deployer) {

deployer.deploy(Calculator);

};

## **🔴 Step 6: Go back**

cd ..

## **🔴 Step 7: Delete old build (PowerShell)**

Remove-Item -Recurse -Force build

## **🔴 Step 8: Compile**

truffle compile

## **🔴 Step 9: Deploy**

truffle migrate \--reset

👉 You should see:

Deploying \'Calculator\'

## **🔴 Step 10: Open console**

truffle console

## **🔴 Step 11: Run commands**

👉 One by one:

let instance = await Calculator.deployed()

await instance.add(5, 3)

let res = await instance.getResult()

res.toString()

## **✅ Output**

\'8\'
