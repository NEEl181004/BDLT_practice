**To install MetaMask and develop a Web3 program using Ethers.js.**

-----
## <a name="_bhys45f1384x"></a>**🔹 PART A: Installation and Setup of MetaMask**
### <a name="_5kr95g3pc746"></a>**Step 1: Install MetaMask Extension**
1. Open Google Chrome
1. Visit[ ](https://metamask.io/)[https://metamask.io](https://metamask.io/)
1. Click on **Download**
1. Select **Add to Chrome**
1. Click **Add Extension**
-----
### <a name="_mta97sfwxgj7"></a>**Step 2: Create a Wallet**
1. Click on the MetaMask icon in the browser
1. Select **Create a Wallet**
1. Set a password
1. Save the **Secret Recovery Phrase** securely
1. Complete the setup process
-----
### <a name="_qzw72g6vee97"></a>**Step 3: Open the Wallet**
Click on **Open Wallet** to access the dashboard-----
### <a name="_wczfd2jhsw91"></a>**Step 4: Select Sepolia Test Network**
1. Click on the network dropdown at the top
1. Select **Sepolia Test Network**
1. If not visible, enable test networks from settings
-----
###
### <a name="_4h0llydw6d6q"></a><a name="_g7tdzp3cypg6"></a>**Step 5: Copy Wallet Address**
1. Click on **Account 1**
1. Copy the wallet address (starting with 0x)
-----
### <a name="_6ffmnm4wndfx"></a>**Step 6: Get Test ETH**
1. Open a Sepolia faucet website

   https://cloud.google.com/application/web3/faucet/ethereum/sepolia

1. Paste the wallet address
1. Request ETH
1. Wait for ETH to be credited to the wallet
-----
