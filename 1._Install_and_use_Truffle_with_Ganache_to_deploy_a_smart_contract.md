# 

**[Install and use Truffle with Ganache to deploy a smart
contract.]{.underline}**

## **🔴 Step 1: Install Node.js (v20)**

-   Install Node **v20 (LTS)**

-   Finish setup

## **🔴 Step 2: Verify installation**

Open **PowerShell**:

node -v

npm -v

✔ Should show v20.x.x

## **🔴 Step 3: Install Truffle**

npm install -g truffle

truffle version

## **🔴 Step 4: Start Ganache**

-   Open Ganache

-   Click **Quickstart**

-   **Keep it running** ✅

## **🔴 Step 5: Create folder**

mkdir MyTruffleProject

cd MyTruffleProject

## **🔴 Step 6: Initialize project**

truffle init

## **🔴 Step 7: Open config file**

notepad truffle-config.js

## **🔴 Step 8: Replace ALL code with:**

module.exports = {

networks: {

development: {

host: \"127.0.0.1\",

port: 7545,

network_id: \"\*\"

}

},

compilers: {

solc: {

version: \"0.8.0\"

}

}

};

## **🔴 Step 9: Go to contracts folder**

cd contracts

## **🔴 Step 10: Create file**

notepad SimpleStorage.sol

## **🔴 Step 11: Paste this:**

// SPDX-License-Identifier: UNLICENSED

pragma solidity 0.8.0;

contract SimpleStorage {

uint public data;

function set(uint x) public {

data = x;

}

function get() public view returns (uint) {

return data;

}

}

## **🔴 Step 12: Go to migrations**

cd ..

cd migrations

## **🔴 Step 13: Create file**

notepad 2_deploy_contracts.js

## **🔴 Step 14: Paste:**

const SimpleStorage = artifacts.require(\"SimpleStorage\");

module.exports = function (deployer) {

deployer.deploy(SimpleStorage);

};

## **🔴 Step 15: Go back**

cd ..

## **🔴 Step 16: Delete old build (PowerShell way)**

Remove-Item -Recurse -Force build

## **🔴 Step 17: Compile**

truffle compile

## **🔴 Step 18: Deploy**

truffle migrate \--reset

## **🔴 Step 19: Open console**

truffle console

## **🔴 Step 20: Run one-by-one**

let instance = await SimpleStorage.deployed()

await instance.set(10)

let value = await instance.get()

value.toString()

## **✅ Output**

\'10\'

# 

# **SIMPLE FLOW**

1.  Install Node

2.  Install Truffle

3.  Open Ganache

4.  Create project

5.  Write contract

6.  Create migration file

7.  Compile

8.  Migrate

9.  Test

  -----------------------------------------------------------------------
  **Command**                    **Why it is used**
  ------------------------------ ----------------------------------------
  node -v                        Checks if Node.js is installed

  npm -v                         Checks if npm (package manager) is
                                 working

  npm install -g truffle         Installs Truffle globally

  truffle version                Verifies Truffle installation

  mkdir MyTruffleProject         Creates a new project folder

  cd MyTruffleProject            Moves into project folder

  truffle init                   Creates project structure (contracts,
                                 migrations, config)

  notepad truffle-config.js      Opens config file to set network and
                                 compiler

  cd contracts                   Goes to contracts folder

  notepad SimpleStorage.sol      Creates smart contract file

  cd ..                          Goes back to main project folder

  cd migrations                  Opens migrations folder

  notepad 2_deploy_contracts.js  Creates deployment file (VERY IMPORTANT)

  Remove-Item -Recurse -Force    Deletes old build files (fixes errors)
  build                          

  truffle compile                Converts Solidity code into bytecode

  truffle migrate \--reset       Deploys contract to Ganache

  truffle console                Opens console to interact with contract
  -----------------------------------------------------------------------
